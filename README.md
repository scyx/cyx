## 快速启动

- 构建工具

  apache-maven-3.6.1

- 开发环境

  JDK 1.8、Mysql 5.7、SpringBoot 1.3.5.RELEASE、redis 5.0.3

在安装之前，需要安装好上述构建工具和开发环境，推荐在linux下安装上述开发环境。

**第一步**；完成数据库的初始化，使用~初始化数据库。

**第二步**；访问项目入口地址

<http://localhost:7777/>
